FAQ and troubleshooting

How do I install a PKHeX plugin?
Installation is very easy. Download and extract the plugin from the zip file into a folder called plugins withing your PKHeX directory.
If you restart PKHeX you will then find a new option in the Tools menu.

PKHeX crashes after I installed your plugin!
Windows 8/10 tends to block dll files you got from the internet for security reasons, but you can easily unblock them.
Right click the plugin, go to Properties and check Unblock in the General tab.
